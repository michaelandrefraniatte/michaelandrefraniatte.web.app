Considerations About the Program

For load local file and create PDF from it, don't name it and place it with names containing spaces. All third libraries must be called with absolute path, present on your computer. Take example on path when you open a file in a web browser, the path is present in the address bar, when you click inside. The program size is made to use pixel measure unity. The local file you load must be in HTML format and with HTML extension.
