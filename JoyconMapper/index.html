Scripting Joycons to Play Games

The program is made to support Joycons in PC games by writting some scripts, you can sale to other users. It's made specificaly for Joycons to not have to write dedicated programs, but there's no input lag, nor drift, and it's totally flawless. Let's see...

Considerations About the Program

The program is made for racing games and indie games. The program is minimized and don't take user by the hand, because I made other programs teaching how scripting, and how it's running. To avoid input lag of Joycon left, connect it after Joycon right, and vice versa. And also run the program with administrative privilege.
