Playtube

Youtube Gallery Player

Keep Tracks of Your Best Videos You are Loving

The program Playtube is a Youtube gallery player where you add videos that can be embed. Make a great list of videos in Playtube for watching it, and share what you add with and in Playtube. Add all videos of a channel in 2 seconds. Or add one by one videos or mix playlists available on Youtube. Enjoy all the power of Youtube. The User Interface (UI) is user friendly for you can change colors as you want. There's nothing better to listen music with video clips of Artists. Let's see...

Considerations About the Program

The folder of the program contains a colors text file for customize it. You can remove a video easy, because when you click on a picture, it copy in clipboard, the picture link containing the video id, and you can open a video in your favorite browser. You can add without any limits. You only need to accept Youtube RGPD. You can try the web version of playtube. I also made the Edge extension named playtube-extension to unzip. The extension can be installed from edge://extensions/. It's made to use on top of Youtube like the program (i.e. opening the extension when your web browser is on Youtube), but it don't allow to cut ads unlike the program. You can try my gallery by opening the example containing my playlists. Take care to always open the extension under same site because it use localstorage for saving the gallery (localstorage info). I made the Edge extension named localstorage-info-extension to unzip. When you let a few times the cursor on buttons, you can see information bubbles, explaining the use of it. This file is explaining the way to get playlist id and channel id, in order to add videos in a playlist using a Youtube data API key. Also, this file containing the id for my example of playlists allowing me to update it. You can download the sdk and program in release folder. My gallery example isn't compatible with the program. With the extension, you can add several videos in a raw using several id of video, channel and playlist in the prompt line by separating with white space. This video can help you to create your Youtube data API key. You can test the process with Listing Videos. To note, with the Edge extension, you can have visualization of audio frequencies by enabling Stereo Mixer, with level to 100 and without listening a peripheral, and disabling microphone from audio settings under Windows parameters. You also must allow Youtube access your microphone (audio input with Stereo Mixer) under Edge. To note, the key F11, under Edge, is used to have the window page in fullscreen. There's a License.
