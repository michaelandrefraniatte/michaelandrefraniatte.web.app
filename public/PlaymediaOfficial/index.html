Playmedia

Media Gallery Player

Enjoy Your Media

The program Playmedia is a video, audio, PDF, image gallery player where you add your media files well place in folders you create. Enjoy your media files in a friendly and nice user interface. You can customize all colors like you want. You can locate each of your media on the world map. Easily save on another disk the media file you are watching or listening. The folders you create, and media files you add in, are automatically loaded. Take nice media files and tidy up it, and then enjoy your media quiet. Let's see...

Considerations About the Program

The thumbnails in the overlay are manually created, names are corresponding to the same names of PDF, video, and audio files with jpg extensions instead of pdf, mp4, and mp3 extensions, and are placed in same roots as PDF, video, and audio files, but it doesn't need to create it for image files. Also Youtube videos can be added using embed files with ytb extension, came from Nash program I made. You can try the web versions for images playshot, and for PDF playbook. You can download the sdk and program in release folder. There's a License.
