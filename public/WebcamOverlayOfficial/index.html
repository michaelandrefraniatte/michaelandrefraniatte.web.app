Webcam Capture as Overlay in Games

Overlay in Your Games with Your Magic Face

The program enable your Webcam to be captured and display the capture as overlay in your games for record you playing. Let's see...

Considerations About the Program

The Numpad keys are allowing to relocate position of overlay, change the size, the transparency, and even make disapearing the overlay. You can change the border by changing the gif file coming with it. It can be a gif file animated. It show also audio visualization coming in your default sound card. It only run with windowed mode of your game, not fullscreen (toggle with Alt+Enter). You can download the sdk and program in release folder, to unzip.
