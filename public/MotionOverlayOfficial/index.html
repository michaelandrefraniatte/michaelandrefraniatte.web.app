Motion Capture as Overlay in Games

Overlay in Your Games with Your Gaming Device Motions

The program enable an overlay for showing gameplay with gaming devices in action, while recording or streaming. Let's see...

Considerations About the Program

The Numpad keys are allowing to relocate position of overlay, change the size, the transparency, and even make disapearing the overlay. It only run with windowed mode of your game, not fullscreen (toggle with Alt+Enter). You can download the sdk and program in release folder. I also made an overlay for the webcam. You can download the sdk and program in release folder, to unzip. You can change the border by changing the gif file coming with it. It can be a gif file animated. It show also audio visualization coming in your default sound card.
