Considerations About the Program

The folder of the program contains a colors text file for customize it. You can add without any limits. You can try the web version of daynote. I made the Edge extension named daynote-extension.zip to unzip. The extension can be installed from edge://extensions/. Take care to always open the extension under same site because it use localstorage for saving tasks and notes. You can download the sdk and program in release folder. There's a License.
