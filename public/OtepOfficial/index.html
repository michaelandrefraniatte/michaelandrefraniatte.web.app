Otep

Gaming Plate for 2 Players

Play at 2 Players Like in Reality

The games of plate allow to play at 2 players. It's easy to get all emotionnals like if you play with another player on real gaming plate. It's for being used with table top, and is optimized for being used with tiny PC. It requires only one Mouse with 2 click buttons and one Keyboard with 2 key buttons to press. Let's see...

Considerations About the Program

The pions can be reinitialized with space bar key, and moved with cursor pressing left click, and also returned with a right click. The escape key is used to close the games. You can try the web version of OtepCheckers, OtepChess, OtepChineseCheckers, OtepOthello, and OtepPower4. For the web version, the key F5 is used to reinitialize the game, the key F11 is used for fullscreen, and the pions are moved with cursor after a simple left click. You can download the sdk and program in release folder. To note, for power 4 plate game, a click on the position where to place the pion is required, it doesn't slice from the top. But unlike the web version I didn't take time to do, the pions can only be placed consecutively like in reality. Also, the programs is only adapted for a resolution of 1366 per 768 pixels.
