Scripting Wiimote and Joycon to Play FPS Games

The program is made to support Wiimote and Joycons in all PC FPS games by writting some scripts, you can sale to other users. It's made specificaly for Wiimote and Joycons to not have to write dedicated programs, but there's no input lag, nor drift, and it's totally flawless. Let's see...

Considerations About the Program

The program is made for being more competitive than usual, but it ask a lot of experiences, and to want aiming and driving like in virtual reality games. The program is minimized and don't take user by the hand, because I made other programs teaching how scripting, and how it's running. To avoid input lag of Joycon left, connect it after Joycon right, and vice versa. And also run the program with administrative privilege.
