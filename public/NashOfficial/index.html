Need All Super Here

Get Audio, Video, Embed Files of Youtube Channels

The program is able to retrieve the videos of a channel or a playlist with link of each video to download it in a raw or one by one. You have the choice between 3 types of files, the embed, the audio and the video. It also download the thumbnail of each video. Let's see...

Considerations About the Program

The program come with another program in order to have the same name of thumbnail and file associate. It also remove the thumbnails if the files associate were not been able to be downloaded. You can download the sdk and program in release folder. This video can help you to create your Youtube data API key, to write in the file apikey.txt under same folder of the program. You can test the process with Listing Videos. There's a License.
