Considerations About the Program

It's easy to understand and make scenes fastly. But I can give my supports like with my other programs. The file arca.txt is used to specify the folder where you put contents. In this folder you put contents, there is a infos.txt file to set the background pictures, the authors, the titles, and the MP3 files of the songs to make the scene. You can use GIF file for background. You can download the sdk and program in release folder.
