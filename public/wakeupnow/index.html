Wakeup

Any Time Any Space

Have the Time Remaining and Be Advertise

The program is for wakeup in music and have time remaining before be alerted. You can change the music, the color of timers, and background image, as you want. If you have something to do important, or somehow forced by something or someone because you engaged you, it's the program to own. Don't be worry you can wakeup with a hunting music if you want, like a bugle, why not? Let's see...

Considerations About the Program

The folder of the program contains a background image file and a colors text file for customize it. You can try the web version of wakeup. I also made the Edge extension named wakeup-extension.zip to unzip. The extension can be installed from edge://extensions/. Take care to always open the extension under same site because it use localstorage for saving the time to wakeup. You can download the sdk and program in release folder. There's a License.
