StopIt

Playing Optimizer

Program to Stop Useless Programs for Playing in Optimized Way

The program is made for stop useless programs. The program itself is optimized. It's used while playing. There's nothing to do, only starting the program. The game you playing is feeling more optimized and so responsive. Let's see...

Considerations About the Program

The program need to set a white list of process to keep running, but it's helped by a record of process stoped by the program. The list is already well completed. It's more simple as it seem to setup, despite with these words. Under same folder of the program you find two text files, the white list to complete and the record of stoped process. Some services like Nvidia container is always trying to start, so it require to disable the service from administrative tools under control panel. You can download the sdk and program in release folder, to unzip. It always need to restart the program after changes in the white list text file. I have a white list text file dedicated to Call of Duty.
