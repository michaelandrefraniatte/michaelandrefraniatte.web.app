MotionInputCore

Scripting Wiimote, Nunchuck, Joycons, Keyboard, Mouse to Play All Games

The program is made to support Wiimote, Nunchuck, Joycons, Keyboard, Mouse in all PC games by writting some scripts, you can sale to other users. It's made specificaly for Wiimote, Nunchuck, Joycons, Keyboard, Mouse to not have to write dedicated programs, but there's no input lag, nor drift, and it's totally flawless. Let's see...

Considerations About the Program

The program is made for being more competitive than usual, but it ask a lot of experiences, and to want aiming and driving like in virtual reality games. To avoid input lag of Joycon left, connect it after Joycon right, and vice versa. And also run the program with administrative privilege and compatibility mode of Windows Vista. You can check how you can use the program with this page: soul-controller. There's a License.
