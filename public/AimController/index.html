Be Like A Pro

Building a Controller with Motion Abilities for PC Gaming

It correspond to a set of dedicated programs for testing controllers with inputs, in order to be plug and play and flawless, receiving data for simulating outputs as a Virtual Xbox Controller or a Virtual Keyboard and Mouse. No input lag, no delay, no stick drift, no gyro drift, no stuttering, fine aiming, adpated and custom control, competitive and enjoyable. You aiming or driving with competitive-immerssive-intuitive-natural-emotional controls. As well as a pro player. Let's see...

Considerations About the Program

The program is made for being more competitive than usual, but it ask a lot of experiences, and to want aiming or driving like in virtual reality games. You can download the sdk and program in release folder. I have programed Teensy, Arduino, Cronusmax, X-Aim, Airmouse, Keyboard, Mouse, Wiimote, Joycons, Pro controller, Dualsense, Xbox controller, PPJoy controller, Six Axis controller, ... for all games. I have lot of knowledge in controllers and motion controls. I have a certain sensitivity for always want the best, the way for it feel good.
