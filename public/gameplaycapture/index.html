GameplayCapture

Capture Your Gameplay

An Optimized Recorder for Your Gameplays

The program capture video and audio of your desktop and is optimized for gameplay capture. The numpad key decimal (.), is used to start the capture, the numpad key 0 is used to stop it. Let's see...

Considerations About the Program

It can crash when the sound card used is from HDMI output audio, but simply the sound card can use a audio peripheral connected with a Wire Jack, to avoid crash. You can download the sdk and program in release folder. I also made a tool to print screen named PrintScreen simply, in a optimized way also. You can download the sdk and program in release folder, to unzip. The print screen key is used for taking screenshots. You shall wait few seconds between each screenshot you taking otherwise you will have a black image. The images are saved under the folder where the program is.
