Merging Sound Tracks and Adjust Timing

This program is allowing to merge audio files in order to have a new audio file. It can be used to merge songs for having an album and adjust the starting time to use in audio or video editors. It's very accurate and easy to use. Let's see...

Considerations About the Program

The file you import must be a MP3 type format file. It can be long to import. You can export the merged file with the name you choose. You can download the sdk and program in release folder, to unzip. There's a License.
