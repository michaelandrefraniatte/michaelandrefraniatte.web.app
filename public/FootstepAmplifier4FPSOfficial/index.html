Considerations About the Program

You shall set the process you want to listen sound, and set the output card to ear sound. It can be used as a virtual bass speaker, enabling to ear sound from multiple speakers. I made the dedicated program for it with the adapted features named VirtualBassSpeaker. You can download the sdk and program in release folder, to unzip. It's made if you have a laptop PC and external speakers as default peripheral for listen music. The internal speakers of your laptop PC are used as bass speaker. Both external and internal speakers must be enable, and level of earing well adapted. If you encounter issue while using the program, you can delete tempsave file under same folder of the program. If you set default speakers and level of earing to 0, the internal speakers of your laptop PC, the program task as an equalizer. By the what, you can have several Bluetooth speakers connected to your laptop PC, and then forward like this the sound by opening several times the program and choosing which speakers to control the sound out.
