Screen Image Saving and Managing

This program is allowing to save your desktop screen as an image. While playing game or watching internet, you can take a screenshot at desire. You can see all your screenshots and delete each screenshot if you don't want to keep it up. Let's see...

Considerations About the Program

The print screen key is used for taking screenshots. You shall wait few seconds between each screenshot you taking otherwise you will have a black image. The images are saved under the folder where the program is.
